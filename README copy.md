# Sacred Heart Secondary School Website

A modern Single Page Application (SPA) built with Vite, React, TypeScript, and Tailwind CSS for Sacred Heart Secondary School.

## Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Modern UI with the school's color palette (#4747d7 primary, #26262c secondary)
- Multiple pages: Home, Academics, Admissions, Apply, Boarding Fees, Activities, Events, Kitchen, Staff, and Contact
- Interactive components including forms, tables, and cards
- Custom icons from Lucide React
- Clean typography using Inter font

## Pages Included

- **Home**: Hero section, principal's welcome, highlights
- **Academics**: Curriculum information and matric results
- **Admissions**: Entry requirements and prospectus download
- **Apply**: Student application form with file upload
- **Boarding Fees**: Fee structure table for 2024/2025
- **Activities**: Image grid for school activities
- **Events**: Upcoming events list
- **Kitchen**: Daily menu cards
- **Staff**: Teacher profile grid
- **Contact**: Contact information and inquiry form

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## Installation

1. Clone the repository or navigate to the project directory
2. Install dependencies:
```bash
npm install
```

## Development

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

To create a production build:
```bash
npm run build
```

## Technologies Used

- React 18 with TypeScript
- Vite as build tool
- Tailwind CSS for styling
- Lucide React for icons
- Responsive design with mobile-first approach

